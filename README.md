## mandeling-ajax
用JavaScript實作非同步「傳送資料給伺服器」與「從伺服器取得資料」雙向網路傳輸模式，即所謂的AJAX．

## 安裝
npm install mandeling

## 引入
### JavaScript
* var ajax = MandelingAJAX();
* var ajax = new MandelingAJAX();
### Vue
* require('mandeling-ajax/mandeling-ajax.js');


## 基本方法
### * get
### * post
### * upload
### * success
### * fail
